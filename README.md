# Si Blogs - 个人博客项目

## Git 提交规范

- `feat` 增加新功能
- `fix` 修复问题/BUG
- `style` 代码风格相关无影响运行结果的
- `perf` 优化/性能提升
- `refactor` 重构
- `revert` 撤销修改
- `test` 测试相关
- `docs` 文档/注释
- `chore` 依赖更新/脚手架配置修改等

## 环境配置说明

**技术栈**
Vite + React + Redux + TS + Less


**下载依赖**

`npm install` 或 `yarn install`

**热重载**

`npm run dev` 或 `yarn dev`

**打包**

`npm run build` 或 `yarn build`
